## Opis projekta
Ovaj projekat je web aplikacija koja omogućava upravljanje korisnicima (Users) koristeći PHP i MySQL.
Aplikacija implementira CRUD operacije:
- **Create** – Dodavanje novog korisnika
- **Read** – Prikaz svih korisnika u tabeli
- **Update** – Izmjena postojećeg korisnika
- **Delete** – Brisanje korisnika

Frontend je urađen koristeći HTML, CSS i Bootstrap, a backend koristeći PHP sa mysqli konekcijom.
Za sigurnost se koriste prepared statements kako bi se spriječio SQL Injection.
